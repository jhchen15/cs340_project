**Citation of Provided Starter Code**
Portions of this project are based on the official CS340 starter application provided by Oregon State University. The starter files include examples of routing, database connection setup, HTML templates, and basic CRUD structure. These files served as the foundation for our application except where otherwise noted in individual source files.
Files derived from the starter application: db_connector.py, athletes.j2, facilities.j2, games.j2, home.j2, main.j2, players.j2, schools.j2, teams.j2, style.css
